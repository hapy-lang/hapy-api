GETTING STARTED

  1. Check if Python is installed globally. If not, download it here
  2. Install Hapy globally. Type in this command in your command prompt:

    pip install Hapy

  3. Then check if the installation was successful by typing this command on a new line:

    hapy --version

  4. Congratulations! Hapy is now installed on your system. Check out example codes here
